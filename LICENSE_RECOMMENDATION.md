# Licence recommendation

No final licence is applied by this package script because third-party data rights must not be overwritten.

Recommended before public release:

- Apply an MIT licence, BSD-3-Clause licence, or equivalent institutional software licence to original code if your institution permits it.
- Apply CC BY 4.0 or CC0 only to author-generated derived tables and figure source files if all authors/institutional policies allow it.
- Do not apply an author licence to reused third-party raw data from IHME, UN WPP, or Natural Earth. Cite those sources and follow their respective access terms.
- If one repository-level licence is required, add a `LICENSE` file plus a `NOTICE` section clarifying that third-party source data remain under their original terms.
